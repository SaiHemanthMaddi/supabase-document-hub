# Media Assets

Store README screenshots and GIFs here.

Recommended filenames:

- `auth-screen.png`
- `dashboard-screen.png`
- `documents-screen.png`
- `profile-screen.png`
- `settings-screen.png`
- `documents-flow.gif`
- `profile-flow.gif`

Tips:

- Use PNG for static screenshots.
- Keep image width around 1600px or less for repo performance.
- Keep GIF files short (<10 seconds) and compressed.
